# Movie App - Приложение для просмотра фильмов
Приложение для просмотра информации о фильмах с использованием React, TypeScript и API Кинопоиска. Проект представляет собой веб-приложение с возможностью поиска, фильтрации, добавления в избранное и сравнения фильмов.

## Основной функционал:
- Просмотр списка фильмов - отображение карточек с постерами, названием, годом выпуска и рейтингом
- Бесконечная подгрузка - автоматическая загрузка новых фильмов при прокрутке страницы
- Фильтрация - по жанрам, рейтингу и году выпуска
- Детальная страница фильма - полная информация: описание, рейтинг, жанры, постер, год выпуска
- Избранное - добавление фильмов с подтверждением
- Сравнение фильмов - возможность сравнивать до 2 фильмов одновременно

## Установка и запуск
### Требования:
- Node.js 18+
- npm 10+
- API ключ от сервиса ПоискКино

### Инструкция по установке:
1. Клонируйте репозиторий
```
git clone https://github.com/TheSun12/movie-app.git
cd movie-app
```
2. Установите зависимости
```
npm install
```
3. Настройте переменные окружения
Создайте файл .env в корне проекта:
```
REACT_APP_API_URL=https://api.poiskkino.dev/v1.4
REACT_APP_API_KEY=ваш_api_ключ
```
4. Запустите приложение
```
npm start
```
Приложение будет доступно по адресу: http://localhost:3000
5. Сборка для production
```
npm run build
```
Собранные файлы будут в папке build/

## Стилизация
Все стили реализованы через объекты CSS-in-JS. Основные цвета:
- Синий (#0077FF) - основной акцент
- Серые оттенки - для текста и фона

## API
Проект использует API ПоискКино. Основные эндпоинты:
- GET	/movie - Получение списка фильмов
- GET	/movie/{id}	- Получение фильма по ID
- GET	/movie/search -	Поиск по названию
